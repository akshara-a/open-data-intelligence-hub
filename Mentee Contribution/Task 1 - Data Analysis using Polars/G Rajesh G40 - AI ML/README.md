# Biodiversity Analytics Engine

## Project Overview

Biodiversity Analytics Engine is a Streamlit-based data analytics platform that analyzes biodiversity data from the IUCN European Red List of Reptiles 2025 dataset.

The application provides insights into species distribution, family classification, conservation status, and endemic species across Europe.

## Features

* Dataset Preview
* Species Analysis
* Family Analysis
* Order Analysis
* European Red List Category Analysis
* Endemic Species Analysis
* Interactive Visualizations
* Species Search Functionality

## Dataset

Source: IUCN European Red List of Reptiles 2025

Dataset File:

* reptiles_data.csv

## Technologies Used

* Python
* Pandas
* Streamlit
* Matplotlib

## Installation

1. Clone the repository

```bash
git clone https://github.com/RAJESH95028/Biodiversity-Analytics-Engine.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```
